## Requirements

* PHP 8.0/8.1
* Node 16

## How to run

```bash
$ cp .env.example .env # create database and import sql file in root folder
$ composer install
$ php artisan migrate
$ php artisan serve
$ npm install
$ npm run dev # compiling asset for development
```

## Compile Assets ( to prod )

```bash
$ npm run build
```
